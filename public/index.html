For the params needed go to: Access->Federation->OAuth Authorization Server->Client Application->
OR
tmsh list apm oauth oauth-client-app "" (this will hold all info for testing purposes)
tmsh list apm profile oauth "" (client-apps will list the ca_list param)
Example:
# tmsh list apm oatuh oauth-client-app oauthTestApp 
            
                    apm oauth oauth-client-app /Common/oauthTestApp { <------ ca_list param
                        app-name "Test API"
                        client-id 1776ade0a3d4be528f3c3728c9520050568666346443565b  <------- client id
                        client-secret 2d36710bb5133b8bdfd5dce4fbde0050568666346443565b <------ client secret
                        customization-group /Common/oauthTestApp_oauth_authz_client_app_customization
                        grant-code enabled
                        redirect-uris { https://www.getpostman.com/oauth2/callback }
                        scopes {
                            /Common/oauth_Scope { } <------ scope_list param
                        }
                    }
